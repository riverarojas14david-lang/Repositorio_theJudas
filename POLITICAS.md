# Políticas del Repositorio

## Uso del repositorio
Este repositorio contiene el desarrollo del proyecto académico "theJudas".

## Reglas de contribución
- Mantener un código ordenado y comentado.
- Realizar commits descriptivos.
- No eliminar archivos sin autorización.

## Seguridad
- No compartir claves privadas.
- No subir información sensible.
## Políticas para Resolución de Conflictos

- Antes de iniciar trabajo en el proyecto, cada integrante debe actualizar el repositorio con:
  git pull origin main

- Los conflictos deben resolverse revisando cuidadosamente los cambios realizados por cada integrante.

- No se deben eliminar cambios sin revisar el impacto en el proyecto.

- Después de resolver un conflicto, se debe verificar que el código funcione correctamente.

- Todo conflicto resuelto debe registrarse mediante un commit descriptivo.

- En caso de conflictos complejos, los integrantes deberán acordar conjuntamente la versión final del archivo.
